# FAiRDAS

This repository provides the code for reproducing the results obtained in the paper *FAiRDAS: Fairness-Aware Ranking as Dynamic Abstract System*. 

## Prerequisites

* Python >=3.9 
* Dependencies:
  ```sh
  pip install -r requirements.txt
  ```
